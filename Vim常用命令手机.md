# Vim常用命令



### 删除

x    		删除光标下的字符 ("dl" 的缩写)
X    		删除光标前的字符 ("dh" 的缩写)
D    		从当前位置删除到行尾 ("d$" 的缩写)
dw   	从当前位置删除到下一个单词开头
db    	从当前位置删除到前一个单词的开头
diw    	删除光标上的单词 (不包括空白字符)
daw    	删除光标上的单词 (包括空白字符)
dG    	删除到文件末
dgg    	删除到文件首
dl  		删除字符 (缩写: "x")         

diW    	删除内含字串 (见 |WORD|)      

daW    	删除一个字串 (见 |WORD|)      

dd    	删除一行               

dis    	删除内含句子           

das    	删除一个句子            

dib    	删除内含 '(' ')' 块           

dab    	删除一个 '(' ')' 块          

dip    	删除内含段落               

dap    	删除一个段落                

diB    	删除内含 '{ ' ' }' 大块            

daB    	删除一个 '{ ' ' }' 大块
